## Executive Summary

**BAIKAL Grant AI**는 정부·공공기관의 보조금 및 지원사업 전 과정(접수→심사→선정→정산)을 디지털화하고, **AI 기술**을 활용하여 심사 효율성과 투명성을 극대화하는 통합 관리 플랫폼입니다.

> 기존의 수작업 중심 보조금 관리 프로세스를 AI 자동 심사와 디지털 워크플로우로 혁신합니다.

## 왜 필요한가?

### 현재의 문제점

| 문제 | 현황 | 영향 |
|------|------|------|
| 🐢 **느린 심사 처리** | 수작업 서류 검토, 평균 8~12주 소요 | 지원자 이탈, 사업 지연 |
| 📑 **비효율적 서류 관리** | 종이·이메일 기반 접수, 분산 보관 | 서류 유실, 추적 불가 |
| ⚖️ **심사 일관성 부재** | 심사자별 주관적 기준 적용 | 공정성 시비, 민원 발생 |
| 🔍 **투명성 부족** | 진행 현황 확인 어려움 | 신뢰도 저하, 문의 폭주 |
| 🚫 **부정 수급 탐지 한계** | 사후 적발 중심 | 예산 낭비, 사회적 비용 |

### BAIKAL Grant AI의 솔루션

```
❌ 수작업 서류 검토          →  ✅ AI 자동 서류 분석 & 점수화
❌ 종이 신청서              →  ✅ 온라인 동적 폼 빌더
❌ 진행상황 불투명           →  ✅ 실시간 상태 추적 & 알림
❌ 주관적 심사              →  ✅ AI 5개 카테고리 정량 평가
❌ 사후 부정수급 적발        →  ✅ AI 사전 리스크 감지
```

## 핵심 기능

### 1. 🤖 AI Smart Review (AI 자동 심사)

AI가 제출된 신청서를 **5개 카테고리**로 자동 분석합니다:

| 카테고리 | 분석 내용 |
|----------|-----------|
| 📋 서류 완성도 | 필수 서류 제출 여부, 기재 내용 충실도 |
| 📊 사업 타당성 | 사업 계획의 논리성·실현 가능성 |
| 🔬 기술 혁신성 | 기술적 차별화, 시장 경쟁력 |
| 💰 재무 건전성 | 재무 상태, 자금 운용 계획 적정성 |
| ✅ 자격 적합성 | 지원 자격 요건 충족 여부 |

**결과 제공**:
- 종합 점수 (0~100점 원형 차트)
- 카테고리별 세부 점수 (수평 바 차트)
- 선정 추천 / 추가 검토 / 반려 권고
- AI 인사이트 (개선 포인트 제시)
- 리스크 평가 (서류누락·자격미달·중복수혜)

### 2. 📝 동적 폼 빌더

사업 담당자가 **코드 없이** 신청서 양식을 자유롭게 설계합니다:

- **7종 필드 유형**: 텍스트, 숫자, 선택, 체크박스, 날짜, 파일, 서약서
- 필드 순서 재배열
- 필수/선택 항목 지정
- 사업별 완전히 다른 양식 가능

### 3. 📊 실시간 대시보드

관리자가 전체 현황을 한눈에 파악합니다:

```
┌────────┬────────┬────────┬────────┬────────┬────────┐
│전체신청 │진행사업│ 선정률  │선정건수 │사용자수 │총 예산  │
│  14건   │  3개   │ 21.4% │  3건   │  8명   │ 4.5억  │
└────────┴────────┴────────┴────────┴────────┴────────┘

📊 상태별 차트  |  🍩 비율 차트  |  📋 사업별 테이블

🔔 최근 활동 피드  |  📄 최근 신청 목록
```

### 4. 🔄 워크플로우 관리

완전한 상태 추적과 이력 관리:

```
신청(임시저장) → 접수 → AI심사/전문가검토 → 선정/탈락/보완요청 → 정산완료
```

- 모든 상태 변경 **자동 기록** (누가, 언제, 왜)
- 사용자에게 **5단계 프로그레스** 시각화
- 상태별 **맞춤 안내 메시지** 제공

### 5. 📄 PDF 생성

신청서를 **공식 서류 형식의 PDF**로 자동 생성:

- 신청자 정보, 답변, 첨부파일 목록 포함
- 관리자·신청자 모두 다운로드 가능
- 접수번호 기반 파일명 자동 생성

### 기술 아키텍처

```
┌─────────────────────────────────────────┐
│           사용자 브라우저                   │
│    React 18 + TypeScript + TailwindCSS   │
│         (프리미엄 반응형 디자인)             │
└──────────────┬──────────────────────────┘
               │ REST API
┌──────────────▼──────────────────────────┐
│         FastAPI Backend (Python)         │
│  JWT 인증 │ AI 심사 │ PDF │ 파일관리     │
└──────────────┬──────────────────────────┘
         ┌─────┼─────┐
         ▼     ▼     ▼
      [  DB  ][Storage][ PDF ]
   PostgreSQL  MinIO   ReportLab
```

### 기술 스택

| 구분 | 기술 |
|------|------|
| **Frontend** | React 18, TypeScript, Vite 5, TailwindCSS 3, Zustand, Recharts |
| **Backend** | FastAPI (Python 3.11), SQLAlchemy, Pydantic v2 |
| **Database** | PostgreSQL 15 (운영) / SQLite (개발) |
| **Storage** | MinIO (S3 호환, 폐쇄망 지원) |
| **인증** | JWT + bcrypt |
| **배포** | Docker Compose (4 서비스) |

## 보안 및 인프라

### 보안 체계
| 항목 | 구현 |
|------|------|
| 인증 | JWT 토큰 기반 인증 |
| 비밀번호 | bcrypt 해시 (salt 자동 적용) |
| 권한 제어 | 역할 기반 접근 제어 (RBAC) |
| API 보호 | 인증 미들웨어, 소유자 검증 |
| 이력 추적 | 모든 상태 변경 워크플로우 로그 |

### 폐쇄망 배포 지원
- 모든 서비스 **Docker 이미지**로 패키징
- MinIO (S3 호환) — 외부 클라우드 없이 파일 저장
- **에어갭(Air-gap) 환경** 배포 가능

```bash
# 이미지 추출 → USB/DVD → 폐쇄망 로드
docker save ... | gzip > baikal-grant-ai.tar.gz
docker load < baikal-grant-ai.tar.gz
docker compose up -d
```

## 시연 가이드

### 빠른 시작

```bash
# 1. 저장소 클론
git clone https://github.com/baikal-ai/baikal-grant-ai.git
cd baikal-grant-ai

# 2. 백엔드 실행
cd backend
pip install -r requirements.txt
python seed_demo.py                    # 데모 데이터 생성
uvicorn app.main:app --reload --port 8000

# 3. 프론트엔드 실행
cd ../frontend
npm install
npm run dev
```

### 시연 시나리오

| 단계 | 역할 | 시연 내용 |
|------|------|-----------|
| 1 | 관리자 | 대시보드 → 6개 통계 카드, 차트, 활동 피드 확인 |
| 2 | 관리자 | 사업 관리 → 신규 사업 생성, 폼 빌더로 양식 설계 |
| 3 | 사용자 | 메인 → 사업 목록 탐색, 상세 보기 (자격요건, FAQ) |
| 4 | 사용자 | 신청 → 폼 작성, 파일 업로드, 최종 제출 |
| 5 | 관리자 | 신청 현황 → 상태 필터, AI 점수 확인 |
| 6 | 관리자 | 상세 심사 → AI 원형 점수, 카테고리 분석, 리스크 확인 |
| 7 | 관리자 | 상태 변경 → "선정" 처리, 워크플로우 이력 확인 |
