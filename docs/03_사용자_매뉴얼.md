## 1. 시작하기

### 1.1 접속 방법
- 웹 브라우저에서 아래 주소로 접속합니다:
  - 개발 환경: `http://localhost:5173`
  - 운영 환경: `http://{서비스주소}`
- 별도의 앱 설치 없이 **PC, 태블릿, 스마트폰** 모두 이용 가능합니다

### 1.2 회원가입

1. 메인 페이지 우측 상단의 **회원가입** 버튼을 클릭합니다
2. 아래 정보를 입력합니다:

| 항목 | 필수 | 설명 |
|------|------|------|
| 이름 | ✅ | 실명 입력 |
| 이메일 | ✅ | 로그인에 사용할 이메일 주소 |
| 비밀번호 | ✅ | 영문, 숫자 포함 8자 이상 권장 |

3. **회원가입** 버튼을 클릭합니다
4. 가입 완료 후 자동으로 로그인 페이지로 이동합니다

### 1.3 로그인

1. 우측 상단의 **로그인** 버튼을 클릭합니다
2. 가입 시 입력한 **이메일**과 **비밀번호**를 입력합니다
3. **로그인** 버튼을 클릭합니다
4. 로그인 성공 시 메인 페이지로 이동합니다

> 💡 **체험용 계정**: 시연 환경에서는 아래 계정을 사용할 수 있습니다
> - 이메일: `hong@startup.kr`
> - 비밀번호: `테스트1234`

### 1.4 로그아웃
- 상단 네비게이션 바의 **로그아웃** 버튼을 클릭합니다

### 1.5 화면 구성

```
┌────────────────────────────────────────────────────┐
│  🏛️ BAIKAL Grant AI  [사업목록] [내 신청] [로그아웃]  │
├────────────────────────────────────────────────────┤
│                                                     │
│              메인 콘텐츠 영역                         │
│                                                     │
│                                                     │
└────────────────────────────────────────────────────┘
```

## 2. 지원사업 찾기

메인 페이지(`/`)에 접속하면 현재 접수 가능한 지원사업 목록을 확인할 수 있습니다.

### 2.1 메인 페이지 구성

#### 🎯 히어로 배너
- **"AI가 지원하는 스마트 보조금 관리"** 메인 타이틀
- 현재 진행 중인 사업 수 표시
- 간단한 서비스 소개

#### ✨ 핵심 기능 안내 (4가지)

| 기능 | 설명 |
|------|------|
| **AI 자동 심사** | 인공지능이 서류를 자동 분석하여 심사 속도 향상 |
| **투명한 프로세스** | 접수부터 결과까지 실시간 상태 확인 가능 |
| **빠른 처리** | 기존 대비 3배 빠른 심사 처리 |
| **간편한 신청** | 온라인으로 간편하게 신청서 작성·제출 |

#### 📋 신청 절차 안내 (4단계)

```
  ① 사업 확인        ② 온라인 신청       ③ AI 심사         ④ 결과 안내
  지원 가능 사업     →  신청서 작성·제출  →  AI 자동 평가   →  선정 결과 확인
  확인 및 선택         파일 업로드          전문가 검토          사업비 지급
```

#### 🟢 진행 중인 사업 목록
- 현재 접수 가능한 사업들이 카드 형태로 표시됩니다
- 각 카드에 표시되는 정보:
  - 사업명
  - 사업 설명 (요약)
  - 접수 기간
  - 지원 금액
  - **"자세히 보기"** 버튼

#### 🔍 사업 검색
- 상단의 검색창에 키워드를 입력하면 사업명으로 실시간 필터링됩니다

#### ⬛ 마감된 사업
- 접수가 종료된 사업도 하단에 별도로 표시됩니다
- 회색 톤으로 구분되며, 상세 정보는 확인 가능하지만 신청은 불가합니다

## 3. 사업 상세 확인

사업 목록에서 **"자세히 보기"** 버튼을 클릭하면 상세 페이지로 이동합니다.

### 3.1 사업 기본 정보

| 항목 | 설명 | 예시 |
|------|------|------|
| 사업명 | 지원사업 전체 이름 | 2026 AI 스타트업 혁신 지원사업 |
| 상태 | 접수중 / 마감 | 🟢 접수중 |
| 사업 설명 | 사업의 목적과 내용 | AI·데이터 기반 혁신 기술 개발... |
| 사업 기간 | 수행 기간 | 2026.04.01 ~ 2026.12.31 |
| 접수 기간 | 신청 가능 기간 | 2026.03.01 ~ 2026.03.31 |
| 지원 금액 | 최대 지원 예산 | 최대 1억원 |
| 심사 방식 | AI + 전문가 복합 심사 | AI + 전문가 복합심사 |

### 3.2 AI 심사 안내
- **"이 사업은 AI Smart Review 시스템이 적용됩니다"** 안내 배너가 표시됩니다
- AI가 제출 서류를 자동 분석하여 심사 효율을 높이고 공정성을 확보합니다
- ⚠️ 지원 자격이 미달인 경우 AI에 의해 자동 반려될 수 있습니다

### 3.3 지원 자격 요건
- 사업별 지원 자격 체크리스트가 표시됩니다
- 예 (AI 스타트업 사업):
  - ✅ AI·데이터 기반 기술을 보유한 스타트업
  - ✅ 법인 설립 후 7년 이내 기업
  - ✅ 상시 근로자 5인 이상
  - ✅ 국내 사업장 보유 기업
  - ✅ 정부 지원사업 수혜 제한 대상이 아닌 기업

### 3.4 신청서 양식 미리보기
- 실제 신청 전 **어떤 항목**을 작성해야 하는지 미리 확인할 수 있습니다
- 각 필드의 이름, 유형, 필수 여부가 표시됩니다

### 3.5 자주 묻는 질문 (FAQ)
- 사업 상세 페이지 하단에 접기/펼치기 형식의 FAQ가 있습니다:

| 질문 | 답변 요약 |
|------|-----------|
| 중복 신청이 가능한가요? | 동일 사업 중복 불가, 다른 사업은 가능 |
| 제출 후 수정할 수 있나요? | 보완 요청 시 수정 가능 |
| 선정 결과는 언제 나오나요? | 접수 마감 후 약 4~6주 |
| 사업비는 어떻게 지급되나요? | 선정 후 협약 체결 → 분할 지급 |

### 3.6 신청하기
- 상세 페이지 하단의 **🚀 "지금 신청하기"** 버튼을 클릭합니다
- 로그인하지 않은 경우 로그인 페이지로 이동합니다
- 접수 마감된 사업은 **"접수 기간이 종료되었습니다"** 메시지가 표시됩니다

## 4. 신청서 작성 및 제출

### 4.1 신청서 작성 화면 접근
- 사업 상세 페이지에서 **"지금 신청하기"** 클릭
- 또는 URL: `/programs/{사업ID}/apply`

### 4.2 작성 방법

사업별로 다른 신청서 양식이 표시됩니다. 각 필드를 순서대로 작성합니다:

#### 텍스트 입력
- 자유롭게 텍스트를 입력합니다
- 필수 항목(*)은 반드시 입력해야 합니다

#### 숫자 입력
- 숫자만 입력 가능합니다
- 예: 매출액, 직원 수, 신청 금액

#### 선택박스 (드롭다운)
- 클릭하여 목록에서 하나를 선택합니다
- 예: 업종, 기업 규모 등

#### 체크박스 (다중 선택)
- 해당하는 항목을 모두 선택합니다
- 예: 보유 인증, 관심 분야

#### 날짜 입력
- 달력에서 날짜를 선택합니다
- 예: 설립일, 사업 시작 예정일

#### 파일 업로드
- **"파일 선택"** 버튼을 클릭하여 파일을 첨부합니다
- 지원 파일: PDF, 한글(HWP), 워드(DOCX), 엑셀(XLSX), 이미지 등
- 선택한 파일명이 버튼 옆에 표시됩니다

#### 서약서 동의
- 내용을 확인하고 **체크박스**를 클릭하여 동의합니다
- 필수 동의 항목은 체크하지 않으면 제출할 수 없습니다

### 4.3 임시저장
- **"임시저장"** 버튼을 클릭하면 작성 중인 내용이 저장됩니다
- 임시저장된 신청서는 **내 신청 현황**에서 "임시저장" 상태로 확인할 수 있습니다
- 이후 다시 수정하여 제출할 수 있습니다

### 4.4 최종 제출

1. 모든 필수 항목(*)을 작성합니다
2. 첨부파일이 있는 경우 파일을 업로드합니다
3. 서약서 동의를 체크합니다
4. **"최종 제출"** 버튼을 클릭합니다
5. 제출 완료 메시지가 표시되고 **내 신청 현황** 페이지로 이동합니다

> ⚠️ **중요**: 최종 제출 후에는 신청서를 직접 수정할 수 없습니다. 관리자가 **보완 요청** 상태로 변경한 경우에만 수정이 가능합니다.

### 4.5 작성 팁
- 📝 모든 내용을 충분히 준비한 후 작성을 시작하세요
- 💾 중간중간 **임시저장**을 활용하세요
- 📎 첨부파일은 가능한 PDF 형식으로 준비하세요
- ✅ 제출 전 모든 항목을 다시 한번 확인하세요

## 5. 내 신청 현황 확인

상단 메뉴의 **"내 신청"**을 클릭하거나 URL `/my-applications`로 접근합니다.

### 5.1 요약 통계 (4카드)

| 카드 | 설명 |
|------|------|
| 📋 **전체 신청** | 지금까지 신청한 총 건수 |
| 🔄 **진행중** | 접수 또는 검토 진행 중인 건수 |
| ✅ **선정/완료** | 선정 또는 정산 완료된 건수 |
| ⚠️ **조치 필요** | 보완 요청된 건수 (빨간 알림 표시) |

### 5.2 신청서 카드

각 신청서가 카드 형태로 표시됩니다:

```
┌──────────────────────────────────────────────────┐
│  📋 APP-0001-20260228-0003                       │
│  접수일: 2026.02.28                    📄 PDF    │
│                                                   │
│  상태: 🟢 검토중                                   │
│                                                   │
│  ── 진행 단계 ──────────────────────────          │
│  ● 신청 ─── ● 접수 ─── ◉ 심사 ─── ○ 결과 ─── ○ 정산  │
│  완료      완료      진행중                        │
│                                                   │
│  💡 AI 심사가 진행 중입니다.                        │
│     심사 중에는 서류를 수정할 수 없습니다.            │
└──────────────────────────────────────────────────┘
```

### 5.3 진행 단계 표시 (5단계 프로그레스)

| 단계 | 설명 |
|------|------|
| ① **신청** | 신청서 작성 (draft) |
| ② **접수** | 최종 제출 완료 (submitted) |
| ③ **심사** | AI + 전문가 심사 진행 (under_review) |
| ④ **결과** | 선정/탈락 결과 (approved/rejected) |
| ⑤ **정산** | 사업 수행 및 정산 완료 (completed) |

- ● (초록 원): 완료된 단계
- ◉ (파란 깜박임): 현재 진행 중인 단계
- ○ (회색 원): 미진행 단계
- ✕ (빨간 점): 탈락 시 표시

### 5.4 상태별 안내 메시지

| 상태 | 메시지 | 색상 |
|------|--------|------|
| 접수 (submitted) | "신청서가 정상적으로 접수되었습니다" | 🔵 파란색 |
| 검토중 (under_review) | "AI 심사가 진행 중입니다" | 🟡 노란색 |
| 보완요청 (revision_requested) | "보완 요청 사항이 있습니다. 확인해 주세요" | 🟠 주황색 |
| 선정 (approved) | "축하합니다! 선정되었습니다" | 🟢 초록색 |
| 탈락 (rejected) | "아쉽지만 이번에는 선정되지 않았습니다" | 🔴 빨간색 |
| 정산완료 (completed) | "사업이 성공적으로 완료되었습니다" | 🟣 보라색 |

### 5.5 PDF 다운로드
- 각 신청서 카드의 **📄 PDF** 버튼을 클릭하면 신청서 PDF를 다운로드할 수 있습니다
- 파일명: `{접수번호}.pdf`

### 5.6 신청서가 없는 경우
- **"아직 신청 내역이 없습니다"** 메시지가 표시됩니다
- **"지원사업 둘러보기"** 링크를 클릭하면 사업 목록으로 이동합니다

## 6. 자주 묻는 질문

### 회원가입 / 로그인

**Q. 회원가입 시 필요한 정보는 무엇인가요?**  
A. 이름, 이메일, 비밀번호만 입력하면 됩니다. 별도의 서류는 필요 없습니다.

**Q. 비밀번호를 잊었으면 어떻게 하나요?**  
A. 현재 비밀번호 찾기 기능은 준비 중입니다. 관리자에게 문의하시기 바랍니다.

**Q. 하나의 계정으로 여러 사업에 신청할 수 있나요?**  
A. 네, 하나의 계정으로 여러 사업에 각각 신청할 수 있습니다.

---

### 지원사업 / 신청

**Q. 동일한 사업에 중복 신청할 수 있나요?**  
A. 동일 사업에 대한 중복 신청은 원칙적으로 불가합니다. 다른 사업에는 별도로 신청 가능합니다.

**Q. 접수 마감일 당일에도 신청이 가능한가요?**  
A. 네, 접수 마감일 자정(24:00)까지 신청이 가능합니다.

**Q. 임시저장한 신청서를 나중에 이어서 작성할 수 있나요?**  
A. 임시저장 기능은 초기 저장 시 사용됩니다. 저장 후에는 내 신청현황에서 확인할 수 있습니다.

**Q. 제출 후 신청서를 수정할 수 있나요?**  
A. 제출 완료 후에는 직접 수정이 불가합니다. 관리자가 **보완 요청** 상태로 변경한 경우에만 수정이 가능합니다.

---

### 심사 / 결과

**Q. AI 심사는 어떻게 진행되나요?**  
A. 제출한 서류를 AI가 자동으로 분석하여 서류 완성도, 사업 타당성, 기술 혁신성 등을 종합 평가합니다. 이후 전문가 검토를 거쳐 최종 결과가 결정됩니다.

**Q. 심사 결과는 언제 나오나요?**  
A. 접수 마감 후 약 4~6주 이내에 결과가 나옵니다. 결과는 **내 신청 현황**에서 확인할 수 있습니다.

**Q. 탈락 사유를 확인할 수 있나요?**  
A. 상태 변경 시 관리자가 남긴 코멘트가 있는 경우, 추후 상세 이력에서 확인할 수 있도록 업데이트 예정입니다.

**Q. 보완 요청을 받으면 어떻게 하나요?**  
A. 신청 현황에서 해당 신청서의 보완 요청 사항을 확인하고, 필요한 서류를 보완하여 다시 제출합니다.

### 지원금 / 정산

**Q. 지원금은 어떻게 지급되나요?**  
A. 선정 후 별도의 협약을 체결하고, 사업 진행에 따라 분할 지급됩니다. 상세 절차는 선정 안내 시 별도 공지됩니다.

**Q. 정산은 어떻게 하나요?**  
A. 사업 종료 후 정산 서류를 제출하면 관리자가 검토 후 **정산완료** 처리합니다.
